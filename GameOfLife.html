Game of Life

A jQuery plugin that does what it says on the tin, namely Conway's Game of Life.

To use, click anywhere on the grid to fill a cell and to unfill a cell click it again.

The buttons Start, Stop, Step and Reset are self explanatory I think :)

Arguments

Initialising this is simple:

The following arguments can be passed:
  • width - the number of cells wide the grid is, default 25.
  • height - the number of cells high the grid is, default 25.
  • tick_length - the time in milliseconds between updates, default 250ms.
  • cell_size - the size in pixels of the cell, default 16.
  • pattern - an array of objects specifying the cell configuration on start up, default null. See below.

pattern

An array of objects, with the the following structure, {StartX:xval, StartY:yval, Data:2darray} that are used to add structures to grid.

StartX and StartY are the grid location where the 2darray contained in data is to be placed. The following example places a vertical bar, 3 cells high, in location 10,10:

{StartX:10, StartY:10, Data:[[0,1,0],[0,1,0],[0,1,0]]}
